⚙️ Setup Guide — 5 steps to enable cloud sync:
1. Go to console.firebase.google.com
2. Click "Add project" → name it (e.g. "tamil-learning") → Create
3. Click "</> Web" icon → Register app → copy the firebaseConfig values
4. In Firebase Console: Authentication → Sign-in method → Enable Google
5. In Firebase Console: Firestore Database → Create database → Start in test mode
Then open this HTML file in a text editor and replace:
apiKey: "YOUR_API_KEY" → your actual key
projectId: "YOUR_PROJECT_ID" → your project id
appId: "YOUR_APP_ID" → your app id
After setup: all users' progress saves to cloud automatically!
